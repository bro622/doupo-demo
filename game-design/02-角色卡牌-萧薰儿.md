# 角色卡牌设计文档：萧薰儿 (Xiao Xun'er)

## 一、 角色概述
* **定位**：高频连击 / 印记引爆 / 极限过牌
* **初始 HP**：65
* **初始能量（斗气）**：3/回合
* **视觉风格**：极简 2D 矢量风。UI 面板采用清透的毛玻璃质感，主色调为耀金（攻击/印记）与青木（身法/过牌）。攻击动作以残影和几何光束为主，不拖泥带水。

---

## 二、 核心机制：【金帝印结】(Golden Seal)

摒弃原版复杂的15层线性加成，将印记改造为类似《英雄联盟》卡莎被动或《原神》元素反应的**"计数引爆"系统**。

* **机制规则**：薰儿的特定卡牌会给敌人施加【金印】。当单个敌人身上的【金印】累计达到 **5 层** 时，印记会立即被**【引爆】**。
* **引爆效果**：清空该敌人身上的所有【金印】，**金帝焚天炎**在目标体内被催化引爆，造成 **10 点真实伤害**（无视护盾），并为你 **返还 1 点能量（斗气）**。
* **设计意图**：玩家的唯一目标变成了"如何在一回合内利用 0 费和 1 费卡牌，精准凑齐 5 的倍数"。这会带来极其流畅的"刷牌"快感和算力的考验。

### 初始遗物：【古族金令】
* **遗物效果**：每回合中，你前两次触发【引爆】时，各抽 1 张牌。每场战斗中，你打出的**第一张攻击牌**额外施加 2 层【金印】。
* *(设计意图：为前期的运转提供基础的过牌保障，确保玩家在第一场战斗即可体验到"引爆回费"的快感，同时鼓励玩家积极去触发印记爆点。)*

---

## 三、 专属关键字

* **金印 (Seal)**：叠加至 5 层时自动【引爆】。视觉上以金印层数标记显示。
* **引爆 (Detonate)**：**金帝焚天炎**在目标体内被催化，触发 5 层金印的特效（10点真伤 + 返还1点能量）。引爆时的视觉特效为金色烈焰从目标体内喷涌而出。
* **连击 (Combo)**：如果你在本回合已经打出过至少 3 张牌，带有此关键字的卡牌会触发额外效果。

> 通用关键字（消耗、力量、敏捷、易伤、虚弱、脆弱、破甲、固有、虚无等）详见 [05-系统策划.md](05-系统策划.md) §二.2。

## 五、 流派连携示例 (Deck archetypes)

1. **无限印记流 (Seal Spam)**：
   * 核心：`神品血脉` + `金焰共鸣` + `印记共鸣` + 大量 0/1 费叠印卡（如 `古族指法`、`流光溢彩`）。
   * 思路：通过 `神品血脉` 降低引爆门槛，使得凑齐 4 层即可返还 1 费并造成真伤。配合遗物的抽牌效果，实现一回合内疯狂叠印、疯狂引爆，能量越打越多，护盾越刷越厚。操作极具节奏感。

2. **帝印溜溜球流 (Yo-Yo Emperor Seal)**：
   * 核心：`帝印决` + `千年传承` + `古族血统` + 精确算牌。
   * 思路：`帝印决` 只要触发引爆就能回手，且引爆本身会返还能量。玩家通过前置卡牌将敌人的印记精确垫到 4 层，然后用 `帝印决` 打出第 5 层，触发真伤、回能、回手，形成完美的局部循环连招。

3. **大寂灭核爆流 (Nuke Finger)**：
   * 核心：`古帝碎涅指` + `印记扩散` + `万印归宗` + `光之终焉`。
   * 思路：平时依靠散碎的印记清理小怪，一旦拖到 Boss 战，在前期苟活阶段疯狂触发引爆，将原本 3 费的 `古帝碎涅指` 减至 0 费，最后一回合配合铺满全场的印记和 `万印归宗` 打出清屏的爆发伤害。

4. **连击节奏流 (Combo Rhythm)**：
   * 核心：`阵法大师` + `凌影突袭` + `掌心雷` + `优雅闪避`。
   * 思路：利用 `阵法大师` 让连击牌无条件触发，配合大量低费牌在一回合内快速堆叠出牌数，打出连击触发的额外金印和护盾，形成攻防一体的节奏感打法。
