# 角色卡牌设计文档：彩鳞 (Cai Lin / Medusa)

## 一、 角色概述
* **定位**：姿态切换 / 毒素控制 / 斩杀吞噬
* **初始 HP**：70
* **初始能量（斗气）**：3/回合
* **视觉风格**：极简 2D 矢量风，带有冷锐切割感的半透明卡面。通过主色调瞬间区分当前姿态：幽紫（女王）与 猩红（吞天蟒）。

---

## 二、 核心机制：【双生姿态】(Dual Stances) 与 【蛇毒】(Venom)

彩鳞可以在两种截然不同的战斗形态之间自由切换，利用"毒素"作为核心资源进行防御或爆发。

### 1. 专属状态：【蛇毒】
* 目标在回合开始时受到等同于当前层数的伤害，随后层数减 1。与萧炎的「燃烧」底层机制完全相同，仅名称和视觉主题不同。彩鳞的毒系卡牌/遗物（如「蟒毒蔓延」「化骨珠」）仅对「蛇毒」生效。

### 2. 姿态切换 (Stance Switch)
战斗开始时，彩鳞默认处于**【无姿态】**。通过打出特定卡牌，可以在以下两种姿态间切换（切换姿态本身不消耗能量）：

* 👑 **【女王姿态】(Queen Form) —— 隐忍、控制与防御**
    * **视觉特征**：角色周身环绕幽紫色的冷光。
    * **姿态被动**：每当你给予【蛇毒】时，额外给予 1 层。你的回合结束时，获得等同于场上最高【蛇毒】层数的护盾。
* 🐍 **【吞天蟒姿态】(Python Form) —— 狂暴、杀戮与吸血**
    * **视觉特征**：角色背后浮现猩红色的巨蟒虚影。
    * **姿态被动**：你的攻击牌会额外造成目标身上【蛇毒】层数的伤害。你受到来自敌人的伤害增加 25%（类似观者的"愤怒"）。

### 3. 初始遗物：【七彩蛇鳞】
* **遗物效果**：每场战斗中，你**首次**切换姿态时，获得 1 点能量，并抽 2 张牌。此外，每场战斗你有 **1 次机会**可以**免费**从任意姿态切回**【无姿态】**（不消耗能量，不占用出牌次数）。
* *(设计意图：鼓励玩家在开局尽快启动姿态引擎，理顺起手牌序。同时提供保底的姿态退出手段，防止因手牌中没有切换姿态的卡而陷入"姿态锁死暴毙"的窘境。)*

---

## 三、 专属关键字

* **进入X姿态**：改变当前形态。如果已经处于该姿态，则不触发任何额外切换效果。
* **吞噬 (Devour)**：如果带有此关键字的卡牌对敌人造成了致命一击（击杀），触发永久性增益（如增加最大生命值）。
* **石化 (Petrify)**：使敌人本回合意图失效（类似强控"眩晕"），极其稀有。
* **连击 (Combo)**：如果你在本回合已经打出过至少 3 张牌，带有此关键字的卡牌会触发额外效果。（与萧薰儿的【连击】为同一通用关键字。）

> 通用关键字（消耗、力量、敏捷、易伤、虚弱、脆弱、破甲、固有、虚无、保留等）详见 [05-系统策划.md](05-系统策划.md) §二.2。

## 五、 流派连携示例 (Deck archetypes)

1. **防毒反伤流 (Queen's Defense)**：
   * 核心：`女王威压` + `幽紫毒雾` + `毒素催化` + `毒蛇体质` + `女王之姿`。
   * 思路：常驻【女王姿态】。利用该姿态"场上最高毒素转化为护盾"的被动，通过翻倍毒素层数，实现单回合自动获得几十甚至上百点护盾。完全放弃主动攻击，看着敌人被毒死并在攻击你时被继续挂毒。

2. **姿态狂舞流 (Stance Dance)**：
   * 核心：`蛇魂共鸣` + `冷血杀手` + `暗影潜行` + `美杜莎之怒` + `姿态精通`。
   * 思路：卡组中塞满大量低费的姿态切换牌。利用切换姿态带来的护盾和力量收益，在一回合内反复横跳。打出 `美杜莎之怒` 时，随机进入一种姿态并触发能力牌被动收益，配合多段伤害打出高额输出。

3. **巨蟒斩杀流 (Python Burst)**：
   * 核心：`狂蟒扫尾` + `致命绞杀` + `生吞活剥` + `蟒毒剧毒`。
   * 思路：极其暴力的打法。前期利用少量卡牌给敌人垫上 4-5 层毒素，然后切入【吞天蟒姿态】，利用攻击牌附加毒素层数的真实伤害，配合减费的 `致命绞杀` 压低血线，最后用 `生吞活剥` 斩杀敌人并永久增加血量，越打越强。

4. **叠毒永续流 (Venom Overflow)**：
   * 核心：`蟒毒蔓延` + `蟒毒体质` + `蟒毒领域` + `蟒毒爆裂` + `蛇魂轮回`。
   * 思路：通过能力牌不断自动叠毒，配合 `蟒毒蔓延` 的额外层数加成，让毒素迅速累积到恐怖的数字。最后用 `蟒毒爆裂` 一次性引爆造成毁灭伤害，或用 `蛇魂轮回` 将毒素转化为续航。
