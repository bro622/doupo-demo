# 守灵系统设计文档 — 各场景 Ancient NPC 设计

> 参考 Slay the Spire 2 的 Ancient Event 系统。每个场景地图的第一个节点为**守灵事件**，进场时**自动满血回复**，随后提供 3 选 1 的恩赐。场景一菩提古树为4选1，场景二至四守灵为3选1。场景一已有「菩提古树」（见 [floor_zero_event.gd](../doupo-demo/scripts/events/floor_zero_event.gd)），本文档补充场景二至四的守灵设计。
>
> **核心设计原则：守灵是叙事的桥接点。** 每个守灵的对话必须承接上一场景的剧情、引出下一场景的冲突，让场景过渡不是"加载新地图"而是"故事的延续"。

---

## 一、叙事桥接总览

### 1.1 场景间的叙事脉络

```
加玛帝国 ──→ 黑角域 ──→ 迦南学院 ──→ 空间虫洞 ──→ 中州
  三年之约     药尘之仇     陨落心炎     空间风暴      魂天帝
  云岚宗灭     韩枫之死     异火争夺     守灵护持      最终决战
               遗物揭秘     古族遗迹
```

每个场景的守灵 NPC 不是随机出现的路人，而是**与该场景核心剧情深度绑定的人物**。他们的对话脚本需要：

1. **回顾**：点出玩家在上一场景完成了什么（击败了谁、获得了什么）
2. **引出**：暗示当前场景将面对什么威胁
3. **选择**：三个选项的文本和效果应带有叙事色彩，而非纯粹的数值交换

### 1.2 三主角各自的叙事线

| 角色 | 守灵人物 | 身份 | 叙事线 |
|:---:|------|------|------|
| **萧炎** | 药尘（药老） | 纳戒中的灵魂导师 | 药尘与韩枫的师徒恩怨 → 药尘指点修炼 → 空间风暴中最后的护持 |
| **萧薰儿** | 古族长老虚影 | 古族血脉的守护者 | 发现古族血脉觉醒 → 传授古族秘法 → 空间风暴中以古族空间之力开路 |
| **彩鳞** | 蛇族先祖 | 远古蛇帝的残念 | 蛇族血脉的共鸣 → 蛇帝圣地的传承 → 觉醒最终姿态 |

**萧炎的药尘线**是最自然的叙事桥——药尘是萧炎的导师，对黑角域（韩枫）、迦南学院（他知道陨落心炎的存在）和中州都有了解。每个场景的守灵对话，本质上就是**药尘在旅途中对萧炎的教导和指引**。场景三→四的空间风暴过渡，是药尘线的高潮——药尘燃烧最后的灵魂力量护持萧炎穿越乱流，既是叙事收束，也为中州的最终决战做好情感铺垫。

**薰儿和彩鳞的线**目前可以先用通用守灵 + 角色专属对话分支来实现，后期再扩展为完全独立的 NPC。见 §六讨论。

## 二、萧炎线 · 药尘的旅途指引

### 叙事逻辑

药尘灵魂寄宿在骨炎戒（萧炎的初始遗物）中。在每个场景过渡时，药尘感应到新区域的气息，主动现身指点萧炎。这不是"随机事件"，而是**师徒之间的对话**——药尘在用自己的经验和记忆帮助萧炎成长。

> **原著改编说明**：原著中萧炎结束"三年之约"后，药老为保护萧炎被魂殿抓走，萧炎在黑角域和迦南学院前期是"单排"状态。游戏中药老全程在线，视为**精简版时间线/平行世界**的改编——在 Roguelike 游戏中，中途将核心向导 NPC 写死会破坏玩家的情感依赖和叙事连贯性。场景四过渡中"药老最后的教诲·印记"暗示药尘将灵魂力量全部注入萧炎，作为叙事收束。

---

### 场景一 → 场景二 · 过渡

**背景**：萧炎击败云山，三年之约尘埃落定。离开加玛帝国，踏入黑角域的荒野。

#### 守灵对话（萧炎专属）

```
药尘的声音从纳戒中传来，语气罕见地凝重——

"云山已死，三年之约已了。你做得很好。"

"但接下来……老夫要告诉你一件事。"
"黑角域里，有一个人——韩枫。"
"他是老夫的叛徒弟子，当年偷走了海心焰。"
"你迟早会与他对上。在那之前……你需要更强。"

"老夫尚有三件事可以指点你。你选一件。"
```

#### 三个选项

| 选项 | 名称 | 效果 | 叙事意义 |
|:---:|------|------|------|
| **A** | **药老指点·净心** | 从牌库移除 2 张基础打击/防御，替换为 2 张随机本职业进阶卡牌（不足则每缺 1 张补 50 金） | 药尘教导萧炎：进入黑角域前，必须精炼卡组，去除花架子 |
| **B** | **药老指点·炼体** | 获得 1 张当前角色随机稀有卡牌 + 100 金币。**代价：失去 10% 最大生命值** | 药尘传授一门高阶秘技，但修炼它需要消耗萧炎的精血 |
| **C** | **药老指点·备药** | 获得 2 瓶随机高级丹药 | 药尘提醒：黑角域凶险异常，韩枫善用毒术，多备丹药保命 |

---

### 场景二 → 场景三 · 过渡

**背景**：萧炎在黑角域艰难击杀韩枫，药尘大仇得报。从韩枫的遗物中，发现了韩枫集结黑角域势力的真正目的——迦南学院底下的异火。

#### 守灵对话（萧炎专属）

```
药尘沉默了很久，灵魂虚影在风中微微颤抖。

"韩枫……死了。"
"老夫等这一天，等了二十年。"
"萧炎，你做得很好。但你看他身上掉落的这份密卷——他在黑角域集结势力，真正的目标居然是迦南学院！"
"迦南学院的天焚炼气塔底下，封印着排名第十四的异火——陨落心炎。韩枫想夺它来补全《焚诀》。"
"异火绝不能落入旁人之手。萧炎，我们立刻前往迦南学院，必须抢在黑角域残党反扑前将它收服！"

"去学院之前，老夫还有三件事可以帮你。"
```

#### 三个选项

| 选项 | 名称 | 效果 | 叙事意义 |
|:---:|------|------|------|
| **A** | **药老传授·秘技** | 获得 1 张当前角色随机稀有卡牌 + 80 金币 | 药尘传授在黑角域战斗中领悟的新技巧 |
| **B** | **药老传授·洗髓** | 从牌库移除 2 张基础打击/防御，替换为 2 张随机本职业进阶卡牌 | 药尘认为：击败韩枫后，萧炎的战斗风格已变，旧的基础牌反而拖累 |
| **C** | **药老传授·古法** | 随机获得 1 个稀有遗物（-8 最大生命值） | 药尘从纳戒深处翻出一件远古遗物，但取出它需要消耗萧炎的精血 |

---

### 场景三 → 场景四 · 过渡

**背景**：萧炎制服陨落心炎，踏入前往中州的空间虫洞。虫洞通道崩塌，遭遇空间风暴。药尘灵魂虚影从骨炎戒中冲出，在乱流中护持萧炎。

#### 守灵对话（萧炎专属）

```
四周的空间通道轰然碎裂，狂暴的银色乱流席卷而来！
药尘的灵魂虚影从骨炎戒中冲出，化作一道淡蓝色的屏障死死护住你。

"空间风暴……小心！"
"萧炎，中州不比加玛帝国，那里是魂殿的大本营，也是真正的修罗场。"
"老夫现在的灵魂力量，只能帮你顶住这阵空间乱流了……接下来的中州之路，以及对抗魂殿的决战，都要靠你自己去闯！"
"在风暴将我们冲散前，接好老夫最后的力量！"
```

#### 三个选项

| 选项 | 名称 | 效果 | 叙事意义 |
|:---:|------|------|------|
| **A** | **药老最后的教诲·顿悟** | 随机升级牌库中的 2 张卡牌 | 药尘在空间乱流的极限压迫下，强行帮萧炎梳理斗气，激发出卡牌的潜能 |
| **B** | **药老最后的教诲·传承** | 随机获得 1 个稀有遗物（-10% 最大生命值） | 药尘将纳戒深处的遗物抛给萧炎，但这让他的灵魂屏障变弱，萧炎被乱流重创 |
| **C** | **药老最后的教诲·印记** | 本局永久 +1 力量 | 药尘在被空间风暴吞噬前，将最后的灵魂印记强行烙入萧炎体内（暗示药尘可能无法再以灵魂形式存在） |

## 三、萧薰儿线 · 古族血脉觉醒

### 叙事逻辑

薰儿拥有远古八族之一"古族"的神品血脉。在旅途的每个阶段，古族血脉会感应到远古的呼唤，以"古族长老虚影"的形式出现，引导薰儿觉醒血脉中沉睡的力量。

### 场景一 → 场景二

**对话**：
```
你体内的古族血脉突然剧烈跳动。
一道金色的虚影在你面前凝聚——那是一位古族长老的残念。

"后辈……你的血脉比我们预想的更强。"
"前方的路很危险。黑角域的黑暗会侵蚀你的灵魂。"
"让老夫帮你稳固根基。"
```

**选项**（效果与萧炎相同，文本以古族为主题）：

| 选项 | 名称 | 效果 | 叙事意义 |
|:---:|------|------|------|
| **A** | **古族洗礼·净脉** | 移除 2 张基础牌 → 换 2 张进阶牌（不足补 50 金/张） | 古族长老以血脉之力洗炼薰儿的斗技根基 |
| **B** | **古族秘法·觉醒** | 稀有卡 + 100 金，**-10% 最大 HP** | 古族长老强行唤醒血脉中的远古记忆，代价是消耗精血 |
| **C** | **古族遗宝·灵药** | 2 瓶随机高级丹药 | 古族长老从血脉记忆中提取的远古丹方 |

### 场景二 → 场景三

**对话**：
```
古族长老的虚影再次出现，但这次更加清晰。

"你从黑角域的杀戮中走出来了……这片混乱之地的鲜血，反而淬炼了你的血脉。"
"那个被你击败的炼药师（韩枫）不过是个跳梁小丑，但他手里的地图很有意思。他一直觊觎的迦南学院地底，竟然有一处与我古族千丝万缕关联的远古遗迹。"
"真正的历练才刚刚开始。去迦南学院吧，那里的力量，本就该属于拥有神品血脉的你。"

"在此之前，接受古族的传承吧。"
```

**选项**：

| 选项 | 名称 | 效果 | 叙事意义 |
|:---:|------|------|------|
| **A** | **古族传承·武学** | 稀有卡 + 80 金 | 古族长老传授一门血脉武学 |
| **B** | **古族传承·洗髓** | 移除 2 张基础牌 → 换 2 张进阶牌（不足补 50 金/张） | 古族长老以血脉共鸣净化卡组 |
| **C** | **古族传承·遗宝** | 稀有遗物，**-8 最大 HP** | 古族长老从血脉深处取出一件远古遗物 |

### 场景三 → 场景四

**对话**：
```
狂暴的空间乱流即将撕裂飞船，你体内的古族血脉突然爆发出刺眼的金光。
古族长老的虚影在虚空中凝结，抬手间，狂暴的空间之力竟被生生镇压。

"空间通道居然崩塌了……哼，雕虫小技。"
"薰儿小姐，中州局势波谲云诡，魂族蠢蠢欲动，大乱将至。"
"让老夫用古族的空间之力，为您在这乱流中开辟一条坦途！这也是老夫目前能为您做的最后准备。"
```

## 五、场景守灵的通用机制

### 5.1 触发规则

| 规则 | 说明 |
|------|------|
| **出现位置** | 每个场景地图的第一个节点（统一使用 `NodeType.ANCIENT`，替换场景一现有的 `NodeType.FLOOR_ZERO`） |
| **触发方式** | 强制进入，击败上一场景 Boss 后直接进入守灵事件 |
| **满血回复** | 进入守灵事件时，HP 自动恢复至最大值（所有选项通用，不单独说明） |
| **每局一次** | 每个场景的守灵在一次运行中只触发一次 |
| **对话分支** | 根据玩家选择的角色显示不同的对话文本和选项名称 |
| **选项效果** | 三主角的选项**效果相同**（平衡性），但**名称和描述文本不同**（叙事色彩） |

### 5.3 守灵的叙事功能总结

| 功能 | 说明 |
|------|------|
| **回顾** | 点出玩家在上一场景完成了什么（击败 Boss、获得异火等） |
| **引出** | 暗示当前场景的核心冲突和 Boss |
| **情感** | 药尘的师徒情、古族长老的血脉召唤、蛇族先祖的使命传承 |
| **选择** | 三个选项的文本应带有叙事色彩（"药老的最后教诲" 而非 "选择一个奖励"） |
| **满血** | 叙事上可解释为：守灵用灵魂力量治愈了旅者 |

## 六、待讨论项

### 6.1 薰儿/彩鳞的守灵是否需要独立 NPC 资源？

目前设计为：薰儿遇到"古族长老虚影"，彩鳞遇到"蛇族先祖"。这两个 NPC 的视觉形象、对话文本与萧炎的药尘完全不同，但选项效果共享。

**两种实现方案：**

| 方案 | 工作量 | 效果 |
|------|:---:|------|
| **A：三套完整对话** | 高（3 场景 × 3 角色 = 9 套对话 + 3 个 NPC 视觉） | 叙事沉浸感最强，每个角色有完全独立的体验 |
| **B：通用守灵 + 角色对话分支** | 中（3 场景 × 1 套通用流程 + 3 角色 × 对话分支） | 代码复用度高，对话差异通过分支实现 |

**推荐方案 B**：先用一个通用的守灵流程框架，对话文本根据 `PlayerManager.current_character` 分支加载。NPC 视觉可以用同一个"虚影"模板，只改变颜色（药尘=淡蓝、古族长老=金色、蛇族先祖=幽紫）。

### 6.2 满血回复的叙事解释

STS2 中 Neow 会先把玩家 HP 设为 0 再补满（有专属动画）。我们的守灵也需要一个叙事上的解释：

- **药尘**："老夫用残余的灵魂力量为你疗伤。"（淡蓝色灵魂光芒包裹角色）
- **古族长老**："血脉共鸣治愈了你的伤痕。"（金色光芒）
- **蛇族先祖**："蛇族的再生之力流淌过你的身体。"（幽紫光芒）

### 6.3 是否支持非首次游戏的跳过？

STS2 的 Neow 在首次游戏时不可用（需要先完成一局解锁）。我们的守灵是否需要：
- 首次游戏自动触发（引导新玩家）
- 后续游戏可选择跳过（节省老玩家时间）

### 6.4 药尘在场景四的对话是否暗示"药尘消失"？

场景四选项 C "药老最后的教诲·印记" 的叙事暗示药尘将灵魂力量全部注入萧炎。这可以为后续剧情埋下伏笔（药尘是否能在击败魂天帝后复活？）。是否需要在通关动画中回收这条线？

## 八、美术需求

### 8.1 节点图标

每个场景共用同一个守灵节点图标类型（`NodeType.ANCIENT`），视觉区分通过**节点底色**实现：

| 场景 | 场景守灵名称（叙事/配置用） | 节点底色 | 说明 |
|:---:|------|:---:|------|
| 场景一 | 菩提古树 | 墨绿 `Color(0.08, 0.18, 0.10)` | 已实现 |
| 场景二 | 黑角域守灵 | 暗红 `Color(0.18, 0.08, 0.08)` | 呼应黑角域的血腥主题 |
| 场景三 | 迦南学院守灵 | 青蓝 `Color(0.08, 0.12, 0.18)` | 呼应学院的修炼阵纹 |
| 场景四 | 中州守灵（空间风暴） | 银灰 `Color(0.15, 0.12, 0.22)` | 呼应空间虫洞与空间乱流的狂暴能量 |

> **"场景守灵名称"vs"角色 NPC"**：每个场景在配置/叙事脚本中有一个场景守灵名称（如"黑角域守灵"），用于事件 ID 和地图生成。但实际显示给玩家的 NPC 形象取决于角色选择——萧炎看到药尘，薰儿看到古族长老，彩鳞看到蛇族先祖。场景守灵名称是逻辑层，角色 NPC 是表现层。

> **废弃资源**：场景一原有的 `mapnode_floor_zero.png` 图标和 `NodeType.FLOOR_ZERO` 枚举值将在统一为 `NodeType.ANCIENT` 后废弃，保留下向兼容但不再新增引用。

### 8.2 NPC 视觉（角色专属）

守灵 NPC 的外观取决于**玩家选择的角色**，而非场景。三主角的 NPC 采用统一的"虚影"场景模板（`ancient_npc.tscn`），通过脚本动态设置颜色和纹理：

| 角色 | NPC 名称 | 主色 | 形态细节 | 满血动画粒子色 |
|:---:|------|:---:|------|:---:|
| 萧炎 | 药尘 | 淡蓝 | 老者轮廓，灵魂半透明，纳戒光芒 | 淡蓝 |
| 薰儿 | 古族长老 | 金色 | 模糊人形 + 金色血脉纹路 | 金色 |
| 彩鳞 | 蛇族先祖 | 幽紫 | 蛇瞳 + 蛇鳞纹理，半透明蛇尾 | 幽紫 |

> **实现方案**：一个基础"虚影"场景，通过 shader 参数切换颜色和纹理细节。萧炎的药尘在菩提古树事件中已有灵魂色调基础，薰儿/彩鳞的 NPC 只需更换颜色和粒子材质。
